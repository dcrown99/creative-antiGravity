# My Kindle 仕様書

## 概要
My Kindleは、ユーザーがGoogle Driveから直接漫画を読むことができるセルフホスト型の漫画リーダーです。ローカルにダウンロードすることなく、漫画アーカイブ（ZIP/CBZ）を閲覧・読書するためのWebインターフェースを提供します。

## 技術スタック
- **フレームワーク**: Next.js 15 (App Router)
- **UIライブラリ**: React 19, Tailwind CSS v4
- **スタイリング**: Shadcn UI (`@repo/ui` 経由)
- **インフラ**: Docker

## データアクセス
- **ソース**: ホスト上で `rclone` を介してマウントされたGoogle Drive。
- **マウントポイント**: `/mnt/gdrive/Manga` (コンテナパス)。
- **フォーマット**: ZIP, CBZ, およびフォルダ構造をサポート。

## ネットワーク
- **ホストポート**: 3001
- **コンテナポート**: 3001
- **URL**: http://localhost:3001

## 主な機能
- **ストリーミング**: マウントされたドライブから画像をオンデマンドで直接読み込みます。
- **進捗管理**: 読書の進捗状況をローカルに保存します（将来的にはDBへの保存を予定）。
- **レスポンシブ**: タブレットやモバイルでの読書に最適化されています。
