# Maintenance Scripts

このディレクトリには、プロジェクトのメンテナンスや一時的な作業用のスクリプトを格納します。

## 目的

- リファクタリング用スクリプト
- エンコーディング修正スクリプト
- データ移行スクリプト
- 一時的な検証・テストスクリプト

## ルール

- 本番環境では使用しないスクリプトのみ格納
- 作業完了後、不要なスクリプトは削除
- README.mdに各スクリプトの目的を記載
